# Figures: the results story

Rendered by `python -m scripts.make_figures` from the committed measurement JSON in [`../results/`](../results/). Regenerate that data with `scripts/run_measurements.sh`, which trains the multi-agent pool too. The standalone variance/exploitability/tilt results have their own measure scripts (see [../GUIDE.md](../GUIDE.md)). Each `.png` is for the write-up; the matching `.html` is interactive.

**Start with [`exec_summary.png`](exec_summary.png)**: every headline edge with its 95% bootstrap CI (the honest "are the edges real?" view). Narrative: [../THESIS.md](../THESIS.md). Bibliography: [../REFERENCES.md](../REFERENCES.md).

| [`tilt_realdata.png`](tilt_realdata.png) | §real-data | Real-data tilt validation (12,320 sessions, 777,687 hand-rows from 120 PokerStars 25NL files; PHH/Kim 2024, CC-BY-4.0, used for opponent-model validation only). After a ≥10bb loss, 873 real 2009 online players play looser (VPIP +2.8pp) and more aggressively (rate +1.6pp), with both 95% CIs excluding 0. The project's emission-only forward-filter HMM detector registers a resolved P(tilted) separation (+0.011, CI excludes 0); shuffled-label placebos collapse to ~0. A separate Baum-Welch HMM corroborates the phenomenon, beating a 1-state model out-of-sample (held-out LL +681; ΔBIC +1315) with an active regime 1.9× enriched for a recent big loss (4.3% vs 2.3% base). |
| [`tilt_lossvswin.png`](tilt_lossvswin.png) | §real-data | Symmetric within-player control: each player's hand after a ≥10bb loss is compared against their hand after an equal ≥10bb win, matching player, big-pot arousal, and event size so only the swing sign differs. After a loss players are +3.6pp more aggressive and +2.9pp looser than after an equal win (95% CIs exclude 0; Cohen d=0.25/0.14; n=685 matched players). The matched effect is larger than post-loss-vs-baseline because players also tighten after a win, a clean prospect-theory loss-aversion asymmetry. |
